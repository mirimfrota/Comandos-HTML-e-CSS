Introdução aos Comandos

O HTML (HyperText Markup Language) é a linguagem padrão usada para criar páginas da web. Ele permite estruturar o conteúdo, como textos, imagens e links.

O CSS (Cascading Style Sheets) é utilizado para definir o estilo e o layout das páginas da web, permitindo personalizar cores, fontes, tamanhos e espaçamentos.

Tags HTML são comandos básicos usados para estruturar documentos HTML. Exemplos incluem <h1> para títulos, <p> para parágrafos e <a> para links.

O CSS é aplicado aos elementos HTML usando seletores. Por exemplo, o seletor <h1> pode ser usado para estilizar todos os títulos principais da página.

h1 { font-size: 30px; font-family: Arial, sans-serif; color: blue; } h2 { font-size: 20px; font-family: Arial, sans-serif; color: red; } p { font-size: 15px; font-family: "Times New Roman", Times, serif; color: black; } img { max-width: 100%; height: auto; display: block; margin: 20px auto; }
